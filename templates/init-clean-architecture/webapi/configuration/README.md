# Configuration - Configuración de WebApi

## Propósito

Esta carpeta contiene la **configuración de servicios y middleware** de la capa de presentación: Dependency Injection, CORS, autenticación, autorización, Swagger, etc.

## Responsabilidades

1. ✅ Configurar Dependency Injection (DI)
2. ✅ Configurar middleware pipeline
3. ✅ Configurar CORS
4. ✅ Configurar autenticación y autorización
5. ✅ Configurar Swagger/OpenAPI
6. ✅ Registrar servicios de presentación

## Estructura según Framework

El contenido de esta carpeta dependerá del framework elegido. Ver guías de implementación específicas.
